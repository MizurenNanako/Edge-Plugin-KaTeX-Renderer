$$\begin{cases}e^{z}&=\displaystyle \sum_{n=0}^{\infty}\frac{z^{n}}{n!}\\&=\displaystyle 1+z+\frac{z^{2}}{2!}+\cdots+\frac{z^{n}}{n!}+\cdots z\in \mathbb C\end{cases}$$

第二中值定理可以用以下的数理逻辑语言进行表述：

对于任意一个连续函数 $f:[a,b] \rightarrow \mathbb{R}$，如果存在 $y_0 \in [f(a),f(b)]$，那么存在 $c \in [a,b]$，使得 $f(c) = y_0$。

这个表述可以用以下的行间公式表示：

∀f:[a,b]→R,∀y0∈[f(a),f(b)],∃c∈[a,b],f(c)=y0\forall f:[a,b] \rightarrow \mathbb{R},\forall y_0 \in [f(a),f(b)], \exists c \in [a,b], f(c) = y_0∀f:[a,b]→R,∀y0​∈[f(a),f(b)],∃c∈[a,b],f(c)=y0​

其中 $\forall$ 表示“对于任意”，$\exists$ 表示“存在”，$:$ 表示“使得”，$\rightarrow$ 表示“映射到”，$\mathbb{R}$ 表示实数集。
